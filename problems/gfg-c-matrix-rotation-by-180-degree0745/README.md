## Problem Statement

Given a 2D square matrix **mat**[][] of size **n x n**, rotate it by **180** **degrees** without using extra space.

**Note:** You must rotate the matrix** in place **and modify the input matrix directly.

**Examples:**

**Input: **mat[][] =** **[[1, 2],
                [3, 4]]
**Output: **[[4, 3], 
        [2, 1]]
**Input: ** mat[][] = [[1, 2, 3], 
                [4, 5, 6],
                [7, 8, 9]]
**Output: **[[9, 8, 7], 
        [6, 5, 4], 
        [3, 2, 1]]

**Constraints:**
1 &le; n &le; 500
0 &le; mat[i][j] &le; 104
